Steps To Protect Your Computer From Cyber Threats

1. Install a Firewall

2. Create a Boot Disk

3. Install Internet Security Software

4. Backup Everything

5. Install Software Updates

6. Disable Hidden Filename Extension

7. Don't Install/Run Unknown Programs

8. Don't Open Unknown Email Attachments/Links

9. Configure Strict Web Browser and Email Security Settings

10. Turn Off Computer and Disconnect from the Internet
